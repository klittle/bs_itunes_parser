count_unique_values_for_key(a_key)

Returns the integer number of unique values in the library for a key such as ‘artist’, ‘album’, or ‘genre’. Sample: count_unique_values_for_key(‘album’) returns 227

find_most_played_songs()

find_playlists_for_song(song_name)

find_recent_songs()

Returns an array of the names of songs most recently modified

find_songs_for_key_value(a_key, a_value)

Returns an array of songs that match a_key and a_value. a_key must match exactly. a_value will match songs that include the string "a_value", and is case insensitive. Sample: find_songs_for_key_value(‘artist’, ‘ryso’) finds songs by "Peabo Bryson"

find_songs_without_key(a_key)

Returns an array of songs that do not contain a_key.
